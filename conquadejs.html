conquadejs

ExpressJS that caters to older technology by generating HTML on the fly.

Here at voga we have been using the ExpressJS webserver to host allmost all of our websites through the official github website serviec that assists in the publishing via DNS. Over time we realised that even though it was a nice system, and used pug it never allowed older browsers (specificly browsers from 1995-2007) to get all of the parsers and serving abilities that is has into action. And thats why we made conquade.

Some features

Fully supportive of multiple browsers, including but not limited to Safari, Google Chrome/Chromium, Oprea, Murcuy, Mozilla Firefox and Tor Browser

Saves time with a start process that takes less than fifteen seoconds to start on the oldest machines.

Runs on any device you can find that can run node and has an internet connection, thanks to the flexibility of node.js
